## czh-project 项目简介
czh-project 是一款帮助Java微服务整合流行项目的开源系统，采用流行的技术，如 SpringBoot、MyBatis、Redis、 MySql、 MongoDB、 RabbitMQ、Elasticsearch，采用Docker容器化部署。
